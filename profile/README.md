# TaskExplorer Software — Portable Advanced Activity Monitor

## About TaskExplorer Software

**TaskExplorer Software** provides a complete advanced process monitor with code signature verification — for system administrators who need deeper insight than built-in Activity Monitor. **TaskExplorer Software** combines process inspection, signature checking, and network monitoring into one cohesive interface. **TaskExplorer Software** supports VirusTotal integration, library dependency viewing, and file handle inspection. **TaskExplorer Software** includes process ancestry, memory usage details, and sandbox status. **TaskExplorer Software** enables administrators to see which processes are signed, unsigned, or potentially malicious. **TaskExplorer Software** is trusted by system administrators and security professionals.

## Deep Dive — TaskExplorer Software Notes

Process list — shows all running processes with details
Code signature — shows if process is signed by Apple or developer
VirusTotal — check process hash against 70+ antivirus engines
Network connections — see which processes are talking to internet
Libraries — view loaded dylibs and frameworks
File handles — see which files process has open
Process ancestry — see parent-child process relationships

## Quick Reference for TaskExplorer Software

| Operation | TaskExplorer Software Action |
|-----------|------------------------------|
| Refresh process list | Click Refresh button or press Cmd+R |
| View process details | Click on process in list |
| Check VirusTotal | Right-click process > VirusTotal |
| View network connections | Click Network tab |
| View loaded libraries | Click Libraries tab |
| View file handles | Click Files tab |
| View process ancestry | Click Parent/Child tab |
| Search processes | Type in search field |
| Export report | File > Export > JSON or CSV |
